## Overview
This project is my solution to the Frontend Mentor Four Card Feature Section challenge. The goal was to recreate the provided design while practicing modern CSS layout techniques and responsive design principles.

### Development process
I used Flexbox to center the overall page layout and CSS Grid to build the four-card section. The cards were arranged using grid-template-areas, which made it easier to position each card according to the design requirements. I also added media queries to adjust the layout for smaller screen sizes and improve responsiveness.

### Challenges faced
The biggest challenge was making the layout responsive. While the desktop version came together fairly quickly, adapting the grid structure for tablets and mobile devices required several adjustments. Managing card widths, spacing, and maintaining the intended visual hierarchy across different screen sizes took a lot of trial and error. Although the result works well, I still feel there is room for improvement in the responsiveness.

### Key learnings
- Improved my understanding of CSS Grid and grid areas.
- Learned when to use Flexbox versus CSS Grid.
- Gained more experience with responsive design and media queries.
- Practiced structuring layouts using modern CSS techniques.
- Better understood how layout decisions affect different screen sizes.
